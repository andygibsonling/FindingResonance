# FindingResonance
A tool for singers and phoneticians to explore formant-tuning

# Instructions
1. download FindingResonance downnload.zip
1. unzip and open praat script
1. run the script and drag your FindingResonance downnload folder to the first field to give it the path
1. record a sound manually. go to praat objects window: New>Record mono sound. Record, then 'save to list'
1. once your sound is selected, go back to prompt and press continue and follow instructions there

1. open and run the R script
1. go to the 'archive' folder and see the folder with current datestamp for wavs, spectra, csv, png
